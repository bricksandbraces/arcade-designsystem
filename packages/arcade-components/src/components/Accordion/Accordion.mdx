# Accordion

The Accordion component is a UI element that expands to reveal hidden content when clicked on. It's useful for hiding and showing large amounts of content within a limited space.

## Usage

To use the Accordion component, import it from its module and include it in your JSX:

```jsx
import { Accordion } from "@openbricksandbraces/designsystem";

<Accordion title="Section 1">This is the content of section 1.</Accordion>;
```
